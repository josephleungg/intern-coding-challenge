# intern-coding-challenge

### To install and test
```bash
# Clone this repository
$ git clone https://github.com/josephleungg/intern-coding-challenge

# Go into server folder
$ cd intern-coding-challenge/server

# install necessary packages
$ npm install # using npm
$ bun install # using bun
$ pnpm install # using pnpm
$ yarn install # using yarn

# run the server (server is running on port 5000)
$ npm run start
$ bun run start
$ pnpm run start
$ yarn run start

# Open a new terminal and go into client folder
$ cd intern-coding-challenge/client

# install necessary packages
$ npm install # using npm
$ bun install # using bun
$ pnpm install # using pnpm
$ yarn install # using yarn

# run the development server (client is running on port 3000)
$ npm run dev
$ bun run dev
$ pnpm run dev
$ yarn run dev

# Go to localhost:3000
```
